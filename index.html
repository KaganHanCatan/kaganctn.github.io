HR Correlation Analytics: Correlation Dashboard Revealing Turnover Dynamics

Some crucial patterns may go unnoticed amid the complexity and intensity of business processes. Overlooking such aspects in the field of human resources can lead to the loss of valuable employees. In this context, I developed the correlation page within the Peopleoma Human Resource Analytics tool to understand and utilize variables associated with employee turnover. This module, designed with a user-friendly interface, aims to be comprehensible even to individuals not specialized in this domain.

Web Application for Statistical Analysis and Machine Learning Algorithms with R and Python Programming Languages

Machine learning algorithms are powerful tools used extensively to facilitate decision-making, especially when the dependent variable is dependent on many numerical or categorical variables. In this project, I created a web page that the user can upload the desired datasets, presenting the descriptive statistics selected according to the user's demand and the dependent variable predicted with a machine learning algorithm chosen by the user. The hyper-parameters can be optimized, and in addition to these, the user can specify the analysis. It is planned to create an interface that presents the results (prediction values, confusion matrix, accuracy, etc.).

Gender Gap Forecast

Efforts to achieve gender equality and fair representation of all genders have become a significant priority in today's business world. In this context, the module I have developed has the capability to use past data to make predictions for the next three years, enabling companies to reach their predefined gender diversity goals. It forecasts the number of individuals to be employed and their gender distribution.
